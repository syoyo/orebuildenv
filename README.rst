orebuildenv
===========

Selfish cross platform build environment.

Supported platforms
-------------------

* Linux 64bit
* Solaris 64bit
* Mac OS X 10.6 64bit
* Windows 64bit

Tools
-----

* premake4(4.4-beta4)
* node.js(0.6.14, 0.6.15)
* ispc(1.2.1)

binary
------

Prebuild binaries are placed at build/ directory.


rebuilding
----------

Linux
~~~~~~~~~~~~~~~~

::

 $ ./build_linux.sh
