orebuildenv
===========

Selfish cross platform build environment.

Supported platforms
-------------------

* Linux 64bit
* Solaris 64bit
* Mac OS X 10.6 64bit
* Windows 64bit

Tools
-----

* premake4

binary
------

Prebuild binaries are placed at build/ directory.
